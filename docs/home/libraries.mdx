We have created a few libraries to help you integrate with the Junction API. We'll continue to add support for more libraries as we add more devices. Email [support@junction.com](mailto:support@junction.com) for specific library support.

### Typed Bindings of Junction API

These typed bindings track the Junction API OpenAPI Schema.

|                                                                      |                                                      |
| -------------------------------------------------------------------- | ---------------------------------------------------- |
| [junction-java](https://github.com/junction-api/junction-java)             | Junction API Java client                                    |
| [junction-go](https://github.com/junction-api/junction-go)                 | Junction API Go client                                      |
| [@junction-api/sdk](https://www.npmjs.com/package/@junction-api/sdk) | Junction API TypeScript client                              |
| [junction-api-sdk](https://pypi.org/project/junction-api-sdk)        | Junction API Python client                                  |
