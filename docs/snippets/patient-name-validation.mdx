## Patient Name Validation

Patient names must conform to the following regex pattern due to lab restrictions:

```
^([a-zA-Z0-9]{1})([a-zA-Z0-9-.,']*(\s[a-zA-Z0-9-.,']+)*[a-zA-Z0-9-.,']?)$
```

**Valid Examples:** `John`, `Mary-Jane`, `O'Connor`, `Dr. Smith`, `Jean-Pierre`

**Invalid Examples:** ` John` (starts with space), `John ` (ends with space), `-John` (starts with hyphen), `John@Smith` (contains @), `Jos√©` (contains accented character)

For complete validation rules and examples, see [Patient Name Validation](/lab/workflow/order-requirements#patient-name-validation).
