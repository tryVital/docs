---
title: "Debugging"
description: "Debug webhook events using the Junction Dashboard Webhooks section or the Svix API to inspect delivery history and retry failed events."
---

You can debug webhook events in one of two ways: the Dashboard or using the Svix API.

### [Junction Dashboard](https://app.junction.com)

The Junction [Dashboard](https://app.junction.com)
has a Webhooks section that allows you to test and observe webhook events.
