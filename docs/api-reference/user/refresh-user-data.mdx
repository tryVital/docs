---
title: "Refresh User Data"
openapi: "POST /v2/user/refresh/{user_id}"
---

<RequestExample>

```bash Bash
curl --request POST \
     --url {{BASE_URL}}/v2/user/refresh/{user_id} \
     --header 'Accept: application/json' \
     --header 'x-vital-api-key: <API_KEY>'
```

```js JavaScript
const user = await client.User.refresh(user_id);
```

```python Python
user = client.User.refresh(user_id)
```

</RequestExample>
