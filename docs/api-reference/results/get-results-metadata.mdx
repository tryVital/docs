---
title: "Get Results Metadata"
openapi: "GET /v3/order/{order_id}/result/metadata"
---

<RequestExample>

```bash cURL
curl --request GET \
     --url '{{BASE_URL}}/v3/order/<order_id>/result/metadata' \
     --header 'accept: application/json' \
     --header 'x-vital-api-key: {YOUR_KEY}'
```

```python Python
from vital import Client

client = Client(
  api_key=<API_KEY>,
  environment="sandbox",
  region="us"
)

data = client.LabTests.get_results_metadata("<order_id">)
```

```javascript Node
const { VitalClient } = require("Vital");

const client = new VitalClient({
  api_key: <API_KEY>,
  environment: "sandbox",
  region: "us"
});

const data = await client.LabTests.Results.getResultsMetadata("<order_id>");
```

</RequestExample>
