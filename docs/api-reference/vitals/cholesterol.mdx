<RequestExample>

```bash Bash
curl --request GET \
     --url {{BASE_URL}}/v2/timeseries/{user_id}/cholesterol/hdl?start_date="2022-05-01"&end_date="2022-06-01" \
     --header 'Accept: application/json' \
     --header 'x-vital-api-key: <API_KEY>'
```

```python Python
from vital import Client

client = Client(
  api_key=<API_KEY>,
  environment="sandbox",
  region="us"
)

data = client.Vitals.cholesterol(
    type="total",
    user_id=*,
    start_date="2022-05-01",
    end_date="2022-06-01"
)
```

</RequestExample>
