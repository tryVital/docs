<RequestExample>

```bash Shell
curl --request GET \
     --url {{BASE_URL}}/v2/summary/activity/{user_id}/raw \
     --header 'Accept: application/json' \
     --header 'x-vital-api-key: <API_KEY>'
```

```python Python
from vital import Client

client = Client(
  api_key=<API_KEY>,
  environment="sandbox",
  region="us"
)

data = client.Activity.get_raw(
    user_id="*",
    start_date="2022-05-01",
    end_date="2022-06-01"
)
```
