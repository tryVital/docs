<RequestExample>
```bash cURL
curl --request POST \
  --url https://api.tryvital.io/management/v1/org/{org_id}/team_data_pull_preferences/{env}/{region} \
  --header 'X-Vital-Org-Key: <api-key>'
  --data '{
    "team_ids": [
        "<uuid>"
    ],
    "providers": {
        "oura": {
            "default": {
                "historical_days_to_pull": 120
            },
            "resource_overrides": {
                "sleep": {
                    "historical_days_to_pull": 180
                }
            }
        }
    }
}'
```
</RequestExample>

<Warning>
    Setting high values in these settings may result in delays in received data and impact the daily polling due to providers' rate limits.
    We recommend setting the custom historical days ranges only with [BYOO](/wearables/connecting-providers/bring_your_own_oauth) setups.
</Warning>
