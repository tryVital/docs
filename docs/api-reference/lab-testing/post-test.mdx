<RequestExample>

```python Python
from vital import Client

client = Client(api_key, "sandbox")

data = client.LabTests.create_test(
  name="Lipids Panel",
  description="Cholesterol test",
  lab_id=5,
  method="testkit",
  sample_type="dried_blood_spot",
  markers=[160],
)

```
