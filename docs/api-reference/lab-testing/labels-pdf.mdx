---
title: "Get Labels PDF"
description: "Download the shipping labels PDF for a testkit lab test order."
openapi: "GET /v3/order/{order_id}/labels/pdf"
---

<RequestExample>

```bash cURL
curl --request GET \
     --url '{{BASE_URL}}/v3/order/<order_id>/labels/pdf' \
     --header 'accept: application/pdf' \
     --header 'x-vital-api-key: {YOUR_KEY}'
     --output file.pdf
```


</RequestExample>
